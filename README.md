# Deadlock-Builder
Gives you all the details on Deadlock items and heroes. Allowing for the creation of builds using certain heroes, general builds.

## Heroes Data

- Shows all heroes in a gallery format utilizing their image and can search for heroes via their name. 
- On clicking on a hero, it opens a hero details modal where you see all the hero's information.

## Items Data

- Shows all the items like the tab in deadlock. You can filter by category of item (vitality, weapon and spirit) and search for an item via the name.
- On clicking on an item, it opens an item details modal where you can see all the item's information. 

# API

This project utilizes [deadlock-api.com](https://deadlock-api.com/) for all of it's data.
