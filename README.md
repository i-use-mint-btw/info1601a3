# Deadlock-Builder
Gives you all the details on Deadlock items and heroes. Allowing for the creation of builds using certain heroes, general builds and also the favoriting of items and heroes.

## Heroes Data

- Shows all heroes in a gallery format utilizing their image. Can favorite heroes and filter by favorite heroes.
- On clicking on a hero, sends to another window where you see all their info
In this second tab (maybe), allows you to create hero specific builds (may be too much, could just favorite a build the user made for any given hero instead, would be less work)

## Items Data

- Shows all the items like the tab in deadlock. You can filter by favorited items, tier, category of item (vitality, weapon and spirit) and search for an item via the name.

# API

This project utilizes [deadlock-api.com](https://deadlock-api.com/) for all of it's data.
